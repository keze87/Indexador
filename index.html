Info Útil

Archivos necesarios 

Documento

  Archivo de texto a procesar.


Configuraciones


  Archivo que cambia el comportamiento del Parser.

  SALTO_PAGINA: caracter que divide Documento en varias páginas.
  SEPARA_PALABRA: caracter/es que (además de '\n') separa/n las palabras de Documento.

  Ej:

  [SALTO_PAGINA]
  caracter="#"

  [SEPARA_PALABRA]
  caracter=" "
  caracter=","
  caracter="?"
  caracter="."
  caracter=";"
  caracter=":"
  caracter="("
  caracter=")"
  caracter="-"


  Instrucciones


  Archivo que contiene ordenes.

  -cp PALABRA: muestra por pantalla la/s posición/es de PALABRA.
  -rp: muestra por pantalla las palabras ordenadas según su cantidad de apariciones.
  -dg: destruye el glosario.

  Ej:

  -cp Algoritmos
  -rp
  -cp FIUBA
  -dg
  -dg
  -rp

Linux 

Preparación

  #En Terminal

  cd LugarDescarga/

  chmod 777 Linux #Permisos de ejecutable


Modo de ejecución


  ./Linux /ruta/a/Documento /ruta/a/Configuraciones /ruta/a/Instrucciones

Windows 

Preparación

  cmd

  C:\ruta\a\Windows.exe C:\ruta\a\Documento C:\ruta\a\Configuraciones C:\ruta\a\Instrucciones

("Linux" y "Windows.exe" pueden llamarse de otro modo)
